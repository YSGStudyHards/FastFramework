# YyFlight.Core
🚀基于ASP.NET Core 6和ABP Framework（ABP VNext ）搭建的简洁快速开发框架，集成通用RBAC权限管理、统一认证授权、JWT、数据仓储、配置中心、日志系统、ELK等相关功能。
